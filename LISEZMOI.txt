Installation : pas à pas
------------------------

   1. Copier et décompresser l'archive automne-xxx.tar.gz téléchargée depuis http://www.automne-cms.org/download/
   2. Créer une base de données pour Automne (cela se fait en un clic sous phpMyAdmin) ou vérifiez que vous avez les identifiants d'une base de données existante.
   3. Lancer le fichier install.php dans votre navigateur. Le processus d'installation va vérifier votre configuration (cf. http://www.automne-cms.org/requirements/ pour plus de détails), créer les tables dans la base de données et vous permettre de personnaliser votre instance d'Automne. 
	Si vous rencontrez un problème ou si vous avez besoin d'aide, vous pouvez à tout moment nous poser vos questions sur le forum ou nous envoyer un mail à contact[at]automne.ws avec le fichier de diagnostic généré par Automne en cliquant sur 'Besoin d'aide' lors de l'installation.
   4. Si le module CLI de PHP est disponible, choisissez durant l'installation l'option "Script en tache de fond".
   5. Une fois l'installation complétée vous pouvez vous connecter avec l'identifiant 'root' et le mot de passe 'automne', cliquez ensuite sur profil utilisateur pour changer le mot de passe du compte root.
   6. Supprimez les fichiers install.php ainsi que automne-xxx.tar.gz et demo-xx.tgz. Laisser ces fichiers sur un site en production représente une faille de sécurité très importante.

Mettre à jour Automne
---------------------

Pour mettre à jour Automne, il suffit de passer successivement les patches de mises à jour téléchargeables sur http://www.automne-cms.org/download/

Configuration serveur
---------------------

 	- PHP version 5.2 ou supérieur
 	- MySQL version 5.0.x ou supérieur
 	- Apache 1.3.x ou supérieur

Pour de meilleures performances, installez PHP avec le module CLI (Command Line Interface) voir http://php.net/manual/features.commandline.php pour plus d'informations.
